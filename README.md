# COVID-19 Detection using Deep Learning (CNN)

**Overview:-**

This project focuses on detecting COVID-19, Pneumonia, and Normal cases from chest X-ray images using a Convolutional Neural Network (CNN). The model is trained on grayscale radiography images and aims to assist in automated medical diagnosis.

**Features:-**

-Multi-class classification (COVID-19, Pneumonia, Normal)

-Deep Learning using CNN

-Image preprocessing & normalization

-Training & validation performance visualization

-Confusion matrix and classification report

**Dataset:-**

The dataset used:-
COVID-19 Radiography Dataset

Organized into folders:-

         COVID
         
         Normal
         
         Viral Pneumonia
         
**Technology Used**

         Python 

         NumPy & Pandas

         OpenCV

         Matplotlib & Seaborn

         TensorFlow / Keras

         Scikit-learn

**Results:-**

-Accuracy & Loss

-Training and validation accuracy/loss plotted over epochs

-Confusion Matrix

-Visualized using heatmap

-Classification Report

Includes:-

       Precision

       Recall

       F1-score
